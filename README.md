# CCA-F 學習指南 — 繁體中文

第一個繁體中文的 Claude Certified Architect – Foundations (CCA-F) 認證學習指南。

## 內容

- **5 大考試領域** — 每個 Domain 的知識點、技能、考試陷阱，全部繁體中文解說
- **12 題模擬測驗** — 互動式選擇題，含情境描述和詳細解析
- **學習資源整理** — 官方課程、社群教材、繁中資源、付費題庫
- **建議學習路線** — 8 週準備計畫

## 技術

純靜態網站，零框架依賴：
- HTML / CSS / vanilla JS
- 部署於 GitHub Pages

## 本地開發

```bash
# 任意 HTTP server 即可
npx serve .
# 或
python -m http.server 8000
```
